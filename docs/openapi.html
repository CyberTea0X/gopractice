Simple Golang Authorization service (1.0.0)

Download OpenAPI specification:Download

Service that uses JWT refresh and access tokens to authenticate users and simulates sms codes

LoginUser

Logins user

query Parameters
phone
string
Example: phone=+123456789
code
string
Example: code=4921
device_id
integer (DeviceId)
Example: device_id=1

The device ID must be unique for each device the user uses.

Responses

http://127.0.0.1/api/login
